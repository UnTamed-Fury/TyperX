# Changelog

All notable changes to TyperX will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of TyperX typing tutor application
- Support for English and Hindi (Krutidev 10) typing
- Multiple practice modes (time-based, custom text)
- Real-time WPM and accuracy tracking
- Material You design with multiple theme options
- SQLite database for storing typing history
- GitHub Actions workflows for CI/CD

## [1.0.0] - 2024-01-18

### Added
- Complete typing tutor application with English and Hindi support
- Krutidev 10 Hindi keyboard layout implementation
- Real-time WPM calculation and accuracy tracking
- Multiple practice categories (lowercase, uppercase, numbers, mixed)
- Material You design with Catppuccin and Dracula themes
- SQLite database for storing typing history
- Settings screen with customization options
- Daily goal tracking
- Sound and haptic feedback
- GitHub Actions workflows for testing and releasing

### Changed
- Enhanced UI with modern Material 3 components
- Improved database schema with Room
- Better error handling and validation
- Optimized performance for smoother typing experience

### Fixed
- Various UI/UX improvements based on user feedback
- Memory leaks in timer implementation
- Keyboard handling for different input methods
- Database synchronization issues

## [0.1.0] - 2024-01-01

### Added
- Initial project setup with basic typing functionality
- English typing support with QWERTY layout
- Basic WPM calculation
- Simple UI with Material Design
- Local storage for practice history
